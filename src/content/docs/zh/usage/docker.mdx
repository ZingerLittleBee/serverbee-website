## Docker 运行命令

```bash
docker run -d \
  -v /proc:/proc \
  --privileged=true \
  --restart unless-stopped \
  --network=host \
  --name=serverbee-web \
  zingerbee/serverbee-web
```

## 查看容器日志

```bash
docker logs serverbee-web
```
