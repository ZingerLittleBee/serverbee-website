## Features

-   Automatic startup
-   Interactive installation
-   Automatic download of web modules
-   Automatic running of web modules
-   Automatic updating of web modules

## Additional Configuration

### Interactive installation (default)

<CodeBlock>
    
```bash
./serverbee-deploy -i
```

</CodeBlock>

### Using custom ports

<CodeBlock>

```bash
./serverbee-deploy -p 8081
```

</CodeBlock>

### Automatic startup (enabled by default)

<CodeBlock>

```bash
./serverbee-deploy -a true
```

</CodeBlock>

### Disable automatic startup

<CodeBlock>

```bash
./serverbee-deploy -a false
```

</CodeBlock>

### Example

<CodeBlock>

```bash
./serverbee-deploy -p 8081 -a false
```

</CodeBlock>

## Compiling from source code

<CodeBlock>

```bash
cargo build --bin serverbee-deploy --release
```

</CodeBlock>

Then you can find the executable file `serverbee-deploy` in `target/release`.
