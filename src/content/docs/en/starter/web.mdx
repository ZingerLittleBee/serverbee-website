## Features

-   CPU load
-   Average load
-   Memory usage
-   Normal operation, startup time
-   File system loading (and disk usage)
-   Disk I/O statistics
-   Network interface
-   Network traffic statistics
-   Process list
-   Detailed process information
-   Kill process
-   Child processes belonging to the process.

## More Settings

### Use custom port

<CodeBlock>

```bash
./serverbee-web -p 8081
```

</CodeBlock>

## Compile from source code

<CodeBlock>

```bash
cargo build --bin serverbee-web --release
```

</CodeBlock>

Then you can find the executable file `serverbee-web` in `target/release`.
