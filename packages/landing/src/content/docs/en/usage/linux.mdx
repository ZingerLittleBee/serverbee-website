## Command-line installation (without graphical interface)

<Collapse title='need to install wget、unzip'>

    <TwoTabSwitch one='Debian' two='CentOS'>
        <div slot='tab1'>

            ```bash
            apt -y update && apt -y install wget unzip
            ```

        </div>
        <div slot='tab2'>

            ```bash
            yum -y update && yum -y install wget unzip
            ```

        </div>

    </TwoTabSwitch>

</Collapse>

Either `deploy` or `web` can be installed, **choose one to install**

-   deploy (includes web functions and additional features such as **interactive installation**, **automatic startup**, and **automatic updates**)
-   web (needs to be **manually** started and updated)

### Install deploy（**recommend**）
