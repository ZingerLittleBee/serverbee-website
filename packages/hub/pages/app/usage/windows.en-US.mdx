
## Windows Server Autostart Without Logging In

The graphical program cannot start without logging in.

You can download the
[serverbee-web](https://github.com/ZingerLittleBee/server_bee-backend/releases) module, and follow these steps:

1. Open Windows Management Tools - Task Scheduler
2. Right-click on Task Scheduler Library - Create Task

    <img src="https://assets.serverbee.app/docs/win-lauch-1.png" width='600' alt='win-lauch-1' />
3. Set Trigger

    <img src="https://assets.serverbee.app/docs/win-lauch-2.png" width='600' alt='win-lauch-2' />
4. Set Action
