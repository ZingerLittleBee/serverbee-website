# Installation

import { Callout, Steps } from 'nextra/components'


系统要求:
- Git
- Docker, Docker-compose
- OpenSSL（用于生成密码和密钥）

## 一键脚本
```bash copy
bash <(curl -s https://raw.githubusercontent.com/ZingerLittleBee/server_bee-backend/main/script/setup.sh)
```
回答几个问题后，脚本会自动帮你安装服务器。

## 使用 Docker Compose
<Steps>
### 克隆 Github 仓库
```bash copy
git clone https://github.com/ZingerLittleBee/server_bee-backend.git
cd server_bee-backend/docker
```

### 设置环境变量
```bash copy
cp .env.example .env
```

### 运行 docker-compose
```bash copy
docker-compose up -d
```
</Steps>

<Callout type="info">
    如果不需要本地服务器的状态，可以在 `docker-compose.yml` 文件中删除 `web` 服务。
</Callout>

**大工告成!🎉**

## 安装常见问题
### 如何修改环境变量
在 `server_bee-backend/docker/.env` 文件中修改环境变量，然后重新运行**一键脚本**选择 2 更新即可重新安装。

<Callout type="info">
    如果没有使用一键脚本，可以手动重新运行。
    ```shell copy
    docker-compose down
    docker-compose up -d
    ```
</Callout>
