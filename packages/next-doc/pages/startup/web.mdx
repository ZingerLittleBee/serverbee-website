## 功能

-   CPU 负载
-   平均负载
-   内存使用情况
-   正常运行、启动时间
-   文件系统装载（和磁盘使用）
-   磁盘 I/O 统计信息
-   网络接口
-   网络流量统计
-   进程列表
-   进程详细信息
-   杀死进程
-   进程所属子进程

## 更多设置

### 使用自定义端口


```bash
./serverbee-web -p 8081
```


## 从源码编译


```bash
cargo build --bin serverbee-web --release
```


然后你可以在 `target/release` 找到可执行文件 `serverbee-web`
