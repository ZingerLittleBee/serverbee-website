## Docker Run Command

```bash
docker run -d \
  -v /proc:/proc \
  --privileged=true \
  --restart unless-stopped \
  --network=host \
  --name=serverbee-web \
  zingerbee/serverbee-web -p 9527
```
Replace `9527` with any port you want.


## View Container Logs

```bash
docker logs serverbee-web
```
