## Windows Server 在不登录情况下自启动

界面程序无法在不登录的情况下启动

可以下载<a target='_blank' href='https://github.com/ZingerLittleBee/server_bee-backend/releases'>
serverbee-web </a>模块，并且进行如下步骤

1. 打开 Windows 管理工具- 任务计划程序
2. 右键点击任务计划程序库-创建任务
3. 具体设置如下图
