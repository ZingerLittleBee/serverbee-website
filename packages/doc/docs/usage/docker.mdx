    systemctl start docker

    # 启动 docker

    # service docker start

    # 开机自启

    systemctl enable docker.service

    systemctl status docker.service

## Docker 运行命令

```bash
docker run -d \
  -v /proc:/proc \
  --privileged=true \
  --restart unless-stopped \
  --network=host \
  --name=serverbee-web \
  zingerbee/serverbee-web -p 9527
```
将 `9527` 替换为任何你想要的端口


## 查看容器日志

```bash
docker logs serverbee-web
```
