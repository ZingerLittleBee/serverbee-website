:::tip
Linux base 系统可以使用 docker 直接运行 serverbee-web，比如：群晖、QTS、Unraid、PVE、Openwrt 等系统。

其他系统请使用 [二进制文件](/usage/linux) 方式安装。
:::

:::caution
仅建议 Linux base 系统使用 docker，否则你获得的数据将是 docker 镜像本身的数据，而不是你的主机数据。

如果使用系统自带 docker 界面安装，请注意记得挂载目录。
:::

## Docker 运行命令

```bash
docker run -d \
  -v /proc:/proc \
  -v /dev:/dev \
  -v /sys:/sys \
  -v /etc:/etc \
  -v /run/media:/run/media \
  -v /sys/class/net:/sys/class/net \
  -v /usr/lib:/usr/lib \
  --privileged=true \
  --restart unless-stopped \
  --network=host \
  --name=serverbee-web \
  zingerbee/serverbee-web -p 9527
```
将 `9527` 替换为任何你想要的端口

## Docker Compose
```bash
version: '3'
services:
  serverbee-web:
    image: zingerbee/serverbee-web
    command: ["-p", "9527"]
    privileged: true
    restart: unless-stopped
    network_mode: host
    volumes:
      - /proc:/proc
      - /dev:/dev
      - /sys:/sys
      - /etc:/etc
      - /run/media:/run/media
      - /sys/class/net:/sys/class/net
```

## 查看容器日志

```bash
docker logs serverbee-web
```
